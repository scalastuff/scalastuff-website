Getting Started

First, add ScalaBeans dependency to your project.

If you use SBT, add following lines to your project: 

		val scalaStuffRepo = "Scala Stuff Repository" at "https://github.com/scalastuff/maven/raw/master"
  		val scalabeans = "org.scalastuff" %% "scalabeans_2.8.1" % "0.1"

If you use Maven, add following lines to your pom.xml: 

		
		...
			
		  		org.scalastuff
		  		scalabeans_2.8.1
		  		0.1
			
		
		
		...
		
  			scalastuff
  			https://github.com/scalastuff/maven/raw/master

ScalaBeans is cross-compiled to Scala 2.8.0, 2.8.1, 2.9.0RC1

Once dependencies are set up, make simple bean (Note: inner classes are not supported): 

		class Person {
			var firstName = ""
			var lastName = ""
			var dateOfBirth = new java.util.Date(0)
		}
As you can see, we declared all the properties as vars and provided not-null default values. ScalaBeans can work with immutable beans as well, you can read about it in Constructors and Building new Bean instances chapters. Discussion about nulls comes in Serialization chapter. Let's leave it as it is for now and go on with introspection: 
		import org.scalabeans.Preamble._
		
		object ScalaBeansExample extends Application {
			val descriptor = descriptorOf[Person]
			for (property <- descriptor.properties)
				println(property.name + ": " + property.scalaType) 
		}
If you run application above, you will see following output: 
		firstName: String
		lastName: String
		dateOfBirth: Date

That's it. In simple cases there is not much more than that. Read in next chapters about bean properties, constructors, type information and how they can be used in more complex scenarios.
