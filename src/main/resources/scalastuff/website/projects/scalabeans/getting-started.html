Getting Started

First, add a ScalaBeans dependency to your project. If you use SBT, add following lines to your project: 

	val scalaStuffRepo = "Sonatype OSS releases" at "http://oss.sonatype.org/content/repositories/releases"
	val scalabeans = "org.scalastuff" % "scalabeans" % "0.1"

If you use Maven, add following lines to your pom.xml: 

	
	...
		
	  		org.scalastuff
	  		scalabeans
	  		0.1
		
	
	
	...
		
  	  Sonatype OSS releases
  		http://oss.sonatype.org/content/repositories/releases

Once dependencies are set up, make a simple bean (Note: inner classes are not supported): 

	class Person {
		var firstName = ""
		var lastName = ""
		var dateOfBirth = new java.util.Date(0)
	}
As you can see, we declared all the properties as vars and provided not-null default values. ScalaBeans can work with immutable data structures as well, you can read about it in Constructors and Building new Bean instances chapters. Null values are discussed in the Serialization chapter. Let's leave it as it is for now and go on with introspection: 
	import org.scalabeans.Preamble._
	
	object ScalaBeansExample extends Application {
		val descriptor = descriptorOf[Person]
		for (property <- descriptor.properties)
			println(property.name + ": " + property.scalaType) 
	}
If you run the application, you will see following output: 
	firstName: String
	lastName: String
	dateOfBirth: Date
